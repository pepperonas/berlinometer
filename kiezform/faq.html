ROTER TRANSFER-QR CODE

Woher bekomme ich den roten Transfer-QR Code?

Der rote Transfer-QR Code (wasserfeste, laminierte PVC-Karte, 5×5 cm, rot/schwarz, mit Hologramm-Siegel) ist in der KiezForm Schmuckbox enthalten, zusammen mit dem Schmuckstück, Authentizitätszertifikat, Bedienungsanleitung und Blockchain-Info-Karte.

Blockchain-Nachweis

  • Identitätsnachweis über 3D-QR Code (am Schmuckstück)
  • Neuer Transfer-Code wird generiert, alter Code wird ungültig

Transfer-Prozess (Schritt-für-Schritt)

Initialisierung

Aktueller Besitzer scannt roten QR Code, gibt Empfänger-Email/ID ein, System generiert Transfer-Token (24h gültig).

Empfänger-Benachrichtigung

Empfänger erhält Email mit Transfer-Link und Token.

Bestätigung

Empfänger bestätigt innerhalb 24h via Link und Token, Blockchain-Block wird erstellt.

Abschluss

Eigentum wird in Blockchain aktualisiert, alter Token wird ungültig, beide Parteien erhalten Bestätigung.

Was passiert, wenn der Transfer nicht bestätigt wird?

Nach 24h läuft der Transfer-Token ab, kein Transfer erfolgt, Eigentum bleibt beim ursprünglichen Besitzer. Ein neuer Transfer kann initiiert werden.

Sicherheitsmerkmale

  • Blockchain-Verifizierung (unveränderlich)
  • 24h Sicherheitsfenster für Überprüfung/Stornierung
  • Pseudonym-System (USR-XXXXXXXX, DSGVO-konform)
  • Einmalige Transfer-Codes, Email-Verifizierung

Transfer rückgängig machen?

Innerhalb 24h

  • Initiator kann stornieren
  • Empfänger kann ablehnen
  • Support kann helfen

Nach Blockchain-Bestätigung

  • Irreversibel, nur Rücktransfer oder Support-Dispute möglich

Troubleshooting

Transfer abgelaufen: Neuen Transfer starten.

Falscher Empfänger: Vor Bestätigung stornieren.

Email nicht erhalten: Spam-Ordner prüfen, Support kontaktieren.

QR Code nicht lesbar: Beleuchtung/Kamera prüfen, Ersatz beantragen.
