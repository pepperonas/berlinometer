# Raspberry Pi Applications

## Port Assignments

| Port | Application | Description |
|------|-------------|-------------|
| 5000 | philips-hue-controller | Philips Hue smart lighting control |
| 5001 | yamaha-rxv-577-controller | Yamaha RX-V577 receiver controller |
| 5033 | weather-tracker | Weather monitoring and tracking service |

## GPIO Pin Assignments

| Pin | Function | Description |
|-----|----------|-------------|
| GPIO 18 | Temperature & Humidity Sensor | DHT sensor for weather-tracker data collection |
